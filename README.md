# Coffee Machine Project

## Overview
This project is designed to manage a coffee machine with various functionalities including brewing coffee, checking inventory, and maintaining the machine.

## Features
- **Brewing Functionality**: Users can brew different types of coffee.
- **Inventory Management**: Tracks coffee beans, water levels, and milk supplies.
- **Maintenance Alerts**: Notifications for cleaning and maintenance schedules.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/FrontN/Coffee-Machine.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Coffee-Machine
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the coffee machine, run:
```bash
npm start
```
